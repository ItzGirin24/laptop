# Sistem Pendataan Laptop ABBSKP

Aplikasi Manajemen Pengumpulan Laptop Siswa untuk ABBSKP (Akademi Bina Bangsa School of Culinary and Pastry).

## Deskripsi

Sistem ini dirancang untuk mengelola pengumpulan laptop siswa secara efisien, dengan fitur-fitur seperti:
- Dashboard monitoring
- Pendaftaran siswa
- Tracking pengumpulan laptop
- Riwayat pengumpulan
- Manajemen izin
- Import/Export data

## Teknologi yang Digunakan

Proyek ini dibangun dengan teknologi modern:

- **Frontend**: React 18, TypeScript, Vite
- **UI Framework**: shadcn/ui, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: MariaDB, Firebase Firestore
- **Deployment**: PWA (Progressive Web App)

## Persyaratan Sistem

- Node.js (versi 18 atau lebih baru)
- npm atau yarn
- MariaDB (untuk server backend)
- Firebase project (untuk Firestore)

## Instalasi dan Setup

### Frontend

```bash
# Clone repository
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Jalankan development server
npm run dev
```

### Backend

Lihat file README-SERVERS.md untuk instruksi setup server backend.

## Build untuk Production

```bash
# Build aplikasi
npm run build

# Preview build
npm run preview
```

## Deployment

Aplikasi ini dapat di-deploy sebagai PWA dan dapat diakses offline setelah instalasi pertama.
